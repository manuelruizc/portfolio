Mobile player A mobile player to listen and download Youtube videos on mp3, create playlists and listen offline.
Github Demo

This project is made for personal use

This was the first app or mobile app that I did with React/React Native, I'm currently rewriting the codebase with the React hooks API, you can download the apk file here. I made this project to have the ability to listen to youtube videos offline and when locking the phone screen or using other apps.

This app was not created to generate profit or to use as an alternative to the official YouTube channel. This app is intended for personal use and to show software development skills. Is not intended for generationg profit or to use instead of the official YouTube services. I encourage everyone to use the official YouTube services instead of this app as this app is only made to show React Native performance and to show personal skills to employers.
